# KeenKeeper


**Keen Keeper** is a simple app to track your calls, texts, and video interactions with friends in one timeline.


## ✨ Features

### 👤 User Details
Users can view detailed profiles by clicking on any friend’s profile.
![User Profile](https://i.ibb.co.com/Vp5W8YGG/Screenshot-2026-04-15-164025.png)


### 📞 Interaction Options
Users can call, text, or start video calls with friends instantly.

### 🕒 Timeline Tracking
All user interactions (calls, texts, and video calls) are stored in a clean timeline, allowing users to review their communication history easily.
![TimeLine](https://i.ibb.co.com/05nTCMx/Screenshot-2026-04-15-164056.png)
### 📊 Stats Dashboard
A smart analytics section that visualizes overall interactions using a pie chart, showing how users connect with their friends over time.
![stats dashboard](https://i.ibb.co.com/mr8nZJhd/Screenshot-2026-04-15-164116.png)
